KEY DESIGN PRINCIPLES
═══════════════════════════════════════════════════════════════════════════════

1. COLOR-AGNOSTIC
   All typography is separate from color. Colors come from the 8-color palette.

2. FIXED SIZES
   No responsive variants. Same size on mobile, tablet, desktop (intentional).

3. WEIGHT HIERARCHY
   400 (body) → 600 (nav) → 700 (subtitle/button/caption) → 900 (heading)

4. LETTER-SPACING
   Tight (-0.04em to -0.02em) for bold headings
   Normal (0em) for body text readability
   Wide (0.2em) for uppercase nav

5. CONSISTENCY
   All similar elements use the same style across all pages
