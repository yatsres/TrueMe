## Responsive Typography Notes

The current system uses **fixed sizes across mobile and desktop**. This is intentional for consistency. If responsive typography is needed:

```jsx
// Current approach (recommended)
<h2 className="text-trueme-heading">Content</h2>

// Alternative with responsive sizes
<h2 className="text-2xl lg:text-trueme-heading">Content</h2>
```

---

## Color + Typography Separation

All typography styles are **color-agnostic**. Colors are applied separately:

```jsx
// Correct approach
<h2 className="text-trueme-heading" style={{ color: '#ff751f' }}>
  Facilitators
</h2>

// Never include color in typography class
// Colors come from the 8-color palette
```
