# Font Styles for 110_Prices Section

## Card Headers
- **Period** (e.g., "Early Registration", "Second Registration")
  - Font Size: 44px (lg:text-3xl = 32px)
  - Font Weight: 900 (font-black)
  - Letter Spacing: -0.02em
  - Color: #427047 (trueme-green)

## Card Price Numbers
- **Price Amount** (e.g., "850", "700", "1000")
  - Font Size: 48px (text-5xl) / 56px (lg:text-6xl)
  - Font Weight: 900 (font-black)
  - Color: #427047 (trueme-green)

## Card Price Currency
- **Currency Symbol** (€)
  - Font Size: 24px (text-2xl)
  - Font Weight: 700 (font-bold)
  - Color: #427047 (trueme-green)

## Price Labels
- **"Program"** / **"Accommodation & Meals"**
  - Font Size: 14px (text-sm)
  - Font Weight: 600 (font-semibold)
  - Text Transform: uppercase
  - Letter Spacing: 0.025em (tracking-wide)
  - Color: #4B5563 (text-gray-600)

## Card Description Text
- **Deposit Info** (e.g., "€300 deposit to confirm your place...")
  - Font Size: 16px (text-sm) on cards
  - Font Weight: 400 (normal)
  - Line Height: 1.5 (leading-relaxed)
  - Color: #374151 (text-gray-700)

## Card Description Subtext (Bold)
- **Bold Portion** (e.g., "€300 deposit", "Balance due before camp starts")
  - Font Size: 16px (text-sm)
  - Font Weight: 700 (font-bold)
  - Color: #427047 (trueme-green)

## Card Footnote
- **"Special rates for returning teenagers..."**
  - Font Size: 12px (text-xs)
  - Font Weight: 400 (normal)
  - Font Style: italic
  - Color: #6B7280 (text-gray-600)

## Card Buttons (CTA)
- **"BOOK A CALL"**
  - Font Size: 16px (text-trueme-button)
  - Font Weight: 700
  - Line Height: 1.2
  - Letter Spacing: -0.01em
  - Text Transform: uppercase
  - Color: white
  - Background: #427047 (trueme-green)

## What's Included / Not Included Headers
- **"What's Included" / "Not Included"**
  - Font Size: 24px (text-2xl) / 32px (lg:text-3xl)
  - Font Weight: 900 (font-black)
  - Letter Spacing: -0.02em
  - Color: #427047 (trueme-green)

## List Items (Features)
- **Feature Text** (e.g., "Full 6-day programme...")
  - Font Size: 14px (text-sm) / 16px (lg:text-base)
  - Font Weight: 400 (normal)
  - Line Height: 1.6 (leading-relaxed)
  - Color: #1F2937 (text-gray-800)

## Checkmark/Cross Symbol
- **✓ or ✕**
  - Font Size: 20px (text-xl)
  - Font Weight: 700 (font-bold)
  - Color: #a3d4c4 (mint) for checkmark / #ff751f (orange) for cross

## Bottom Section Text
- **"Returning teenagers and siblings..."**
  - Font Size: 18px (text-lg) / 20px (lg:text-xl)
  - Font Weight: 700 (font-bold) for key phrases
  - Line Height: 1.6 (leading-relaxed)
  - Color: white

## Plus/Plus Sign
- **"+"**
  - Font Size: 14px (text-sm)
  - Font Weight: 600 (font-semibold)
  - Color: #6B7280 (text-gray-500)

## VAT Disclaimer
- **"*VAT (23%) not included"**
  - Font Size: 12px (text-xs)
  - Font Weight: 400 (normal)
  - Color: #6B7280 (text-gray-500)

## Summary
All typography uses the **BricolageGrotesque** variable font family with responsive sizing that adapts from mobile to large screens using Tailwind breakpoints.
